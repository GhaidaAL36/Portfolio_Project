# User Stories

## Create Account

| Priority | User Story | # |
| --- | --- | --- |
| Must Have | As a new user, I want to create an account using my email and password so that I can start using the application. | 1 |
| Should Have | As a new user, I want to receive a confirmation message after creating my account so that I know my account was created successfully. | 2 |
| Could Have | As a new user, I want to add optional information like profile picture during registration so that my profile looks complete. | 3 |
| Won't Have | As a new user, I want to sign up using social media accounts so that registration becomes faster. | 4 |

---

## Sign In

| Priority | User Story | # |
| --- | --- | --- |
| Must Have | As a registered user, I want to sign in using my email and password so that I can access my account. | 1 |
| Should Have | As a user, I want to stay logged in so that I do not need to sign in every time I open the app. | 2 |
| Could Have | As a user, I want to sign in using Google or social media so that logging in becomes easier. | 3 |
| Won't Have | As a user, I want to log in using biometric authentication so that access becomes faster. | 4 |

---

## Profile Page

| Priority | User Story | # |
| --- | --- | --- |
| Must Have | As a user, I want to view my profile information so that I can see my personal details. | 1 |
| Should Have | As a user, I want to edit my profile information so that I can update my details. | 2 |
| Could Have | As a user, I want to upload a profile picture and bio so that other users know more about me. | 3 |
| Won't Have | As a user, I want to customize my profile theme so that my page looks unique. | 4 |

---

## Exploring Page

| Priority | User Story | # |
| --- | --- | --- |
| Must Have | As a user, I want to browse the explore page so that I can discover new content. | 1 |
| Should Have | As a user, I want to see recommended content based on my interests so that I find relevant posts. | 2 |
| Could Have | As a user, I want to explore trending topics so that I can see what is popular. | 3 |
| Won't Have | As a user, I want to explore content by location so that I can discover nearby trends. | 4 |

---

## Search Bar

| Priority | User Story | # |
| --- | --- | --- |
| Must Have | As a user, I want to search for users or content so that I can quickly find what I need. | 1 |
| Should Have | As a user, I want to see search suggestions while typing so that I can find results faster. | 2 |
| Could Have | As a user, I want to filter search results so that I can narrow down the results. | 3 |
| Won't Have | As a user, I want to search using voice so that searching becomes easier. | 4 |

---

## Major Info Page

| Priority | User Story | # |
| --- | --- | --- |
| Must Have | As a user, I want to browse all available career simulations so that I can choose the one that interests me most. | 1 |
| Should Have | As a user, I want to see a description of each career including required skills and typical tasks so that I can make an informed choice. | 2 |
| Could Have | As a user, I want to filter careers by field (e.g., Technology, Business, Design) so that I can find relevant options quickly. | 3 |
| Won't Have | As a user, I want to save my favorite careers to revisit later so that I don't have to search again. | 4 |

---

## Task Page

| Priority | User Story | # |
| --- | --- | --- |
| Must Have | As a user, I want to start a job simulation so that I can experience what the role feels like. | 1 |
| Must Have | As a user, I want to see clear instructions before starting a task so that I understand what is expected. | 2 |
| Must Have | As a user, I want to complete tasks that mimic real job responsibilities so that I can evaluate my interest in the career. | 3 |
| Must Have | As a user, I want to submit my work so that it can be evaluated. | 4 |
| Must Have | As a user, I want different types of tasks within the same career so that I see the full picture. | 5 |
| Should Have | As a user, I want to see my progress within a task so that I know how much is left. | 6 |
| Could Have | As a user, I want to save and resume a simulation so that I can complete it across multiple sessions. | 7 |
| Could Have | As a user, I want the AI to adapt the difficulty based on my performance so that the experience feels personalized. | 8 |

---

## Review Page

| Priority | User Story | # |
| --- | --- | --- |
| Must Have | As a user, I want to see my overall performance score after completing the simulation so that I can understand how well I did. | 1 |
| Should Have | As a user, I want to receive detailed feedback on each task I completed so that I know what I did right and wrong. | 2 |
| Could Have | As a user, I want to see a recommended learning plan based on my results so that I know what skills to develop. | 3 |
| Won't Have | As a user, I want to share my review results on social media so that I can show my progress to others. | 4 |
