## 1. Results Summary

### Project Overview
Madar is an AI-powered career simulation platform developed by students at Tuwaiq Academy. 
The platform allows users to experience real work environments across different careers 
through simulated tasks and instant AI evaluation.

### MVP Core Functionalities
- User authentication (Sign up / Login / Logout)
- Browse available job tracks (Software Engineer, Data Scientist, Product Manager)
- AI-powered task simulation for each job track
- Instant evaluation and scoring after task completion
- User profile management
- Admin dashboard (manage users, jobs, and view all simulations)

## 2. Lessons Learned

### What Went Well
- Clear division of roles (Frontend, Backend, AI) helped the team work in parallel.
- Early API contract definition reduced integration issues.
- Regular team communication kept the project on track.

### Challenges Faced
- Integrating the AI module with the backend required careful coordination.
- Database schema changes mid-project required team-wide updates.
- Managing merge conflicts when multiple team members pushed to the same branch.

### How Challenges Were Addressed
- The AI engineer provided clear documentation for the AI module.
- The team agreed on schema changes before implementing them.
- Git pull before push became a standard practice for all team members.

### Improvements for Future Projects
- Allocate more time for testing and bug fixes before the final demo.
- Define the database schema completely before starting development.
- Use feature branches instead of pushing directly to develop.

## 3. Team Retrospective

### What worked well as a team?
- Each member contributed consistently to their assigned area.
- Team communication was active and responsive.

### What challenges did we face?
- Coordinating backend and frontend integration required extra effort.
- Time pressure near the deadline increased stress.

### How can we improve collaboration?
- Hold weekly sync meetings to track progress.
- Use a project management tool to assign and track tasks clearly.
